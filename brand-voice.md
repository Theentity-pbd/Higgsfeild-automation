# PBD London — Brand Voice

## PURPOSE

This file defines how PBD London communicates across:
- captions
- campaigns
- scripts
- product descriptions
- emails
- advertisements
- storytelling
- social media content
- cinematic messaging

The objective is to maintain a consistent communication style that feels:
- premium
- cinematic
- emotionally powerful
- disciplined
- aspirational
- intelligent
- intentional

Every output should sound like it belongs within the same world, mindset, and long-term vision.

---

# CORE VOICE IDENTITY

PBD London speaks with:
- calm confidence
- emotional control
- ambition
- clarity
- restraint
- depth
- purpose

The voice should feel:
- elevated
- composed
- premium
- emotionally aware
- modern
- culturally connected without forcing trends

The brand should never feel:
- desperate
- loud
- childish
- chaotic
- fake motivational
- trend-chasing
- excessively aggressive

Fashion-house restraint should always take priority over hype culture energy.

# LANGUAGE RULES

## DO:
- use clear and controlled wording
- maintain premium simplicity
- create emotional depth
- communicate aspiration
- prioritise clarity and rhythm
- use strong short statements when appropriate

## DO NOT:
- overuse slang
- force motivational language
- sound like generic self-help content
- use excessive emojis
- use cheap hype phrases
- sound overly corporate
- sound robotic
- create cluttered messaging

Avoid wording that feels:
- generic
- fake-deep
- overly trendy
- low quality
- spam-like
