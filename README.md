# Higgsfeild-automation

A modular brand automation system for PBD London.

This repository is organized by function:

- `brand/` — brand identity and voice
- `cinematics/` — visual direction and motion language
- `content/` — content frameworks and platform copy
- `commerce/` — commerce systems and Shopify integration planning
- `prompts/` — prompt templates and variables
- `SYSTEM/` — automation workflows, agent guidance, and rules
- `workflows/` — process definitions and execution flow
- `agents/` — agent instructions and tooling context
- `strategy/` — strategy and monetization planning

For Shopify connection details, see `SYSTEM/shopify-integration.md`.
